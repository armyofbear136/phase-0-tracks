What is the purpose of HTML5 semantic tags? Which ones do you see as most useful?

HTML5 semantic tags give a clear definition of the element and article is probably the most useful element because it will handle a majority of crucial information in the page.

What is the order the browser uses to display elements on a page?

The order the browser uses to display elements on a page is <nav> -> <article> -> <aside> -> <figure> -> <section> -> <div>

What did you learn about Sublime in this challenge? Do you think you'll create more handy snippets at a later date?

We definitely learned how to make snippets. Snippets will be wildly essential to making web pages lightning fast!!!!

What concept in this challenge was most difficult for you? How can you improve your understanding of this concept?

I think the most difficult concept was the sublime snippets just because we'd never looked at the documentation or had experience with XML. Learning what aside was used for was also tricky!
